### Video Request Platform

A single-page application (SPA) where users can submit and vote on video requests for a content creator.

#### Overview

This is a fullstack project built with vanilla JavaScript, Express.js, and MongoDB. It allows content creators and their audience to collaborate on video content through a simple request and voting system.

## Tech Stack

- **Frontend**: Vanilla JavaScript (no frameworks), BootStrap for styling
- **Backend**: Node.js with Express.js
- **Database**: MongoDB
- **Architecture**: MERN-inspired stack (without React)
